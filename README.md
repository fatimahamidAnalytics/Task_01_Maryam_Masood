# Rule-Based AI Chatbot

## Project Description

This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot responds to predefined user inputs by using conditional (`if-elif-else`) statements. It provides basic conversational responses and continues interacting with the user until an exit command is entered.

The purpose of this project is to demonstrate the fundamentals of chatbot development without using Machine Learning or Artificial Intelligence frameworks. It focuses on implementing rule-based logic, user interaction, and basic Python programming concepts.

## Features

- Responds to greetings such as "Hello" and "Hi"
- Answers simple questions like "How are you?" and "What is your name?"
- Displays the current system time
- Tells a programming joke
- Responds to thank-you messages
- Provides a help message
- Handles unknown inputs with a default response
- Allows the user to exit using "bye", "exit", "quit", or "goodbye"

## Technologies Used

- Python 3
- Built-in `datetime` module

## How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

5. Start chatting with the bot.
6. Type `bye`, `exit`, or `quit` to end the conversation.
